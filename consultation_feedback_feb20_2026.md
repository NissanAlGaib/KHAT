# Instructor Feedback – Admin Side

## Bug Fixes / UI Improvements

### 1. Report/Ticket Action Buttons
- Simplify the action buttons to just **two options**: `Resolved` and `Dismissed`.

### 2. Date Filter
- Add a **date range filter** that includes options for: **Today**, **This Year**, and **Last Year**.

---

## Feature Suggestions

### 3. Discounts & Promotions per Company
- Allow admins to set a **discount or sale percentage per company**.
- Support different promo types: **sale prices**, **voucher codes**, and **promo codes per user**.

### 4. Admin Audit Log
- Admins should be able to **see a history of changes**—specifically when another admin has changed or reset something (e.g., passwords, settings).

### 5. Time Travel / Expiry Testing
- Add a **"time travel" feature** that lets admins simulate a future date to **test how expiring items behave** (e.g., promos, subscriptions, vouchers).

### 6. Revenue Drill-Down
- When an admin clicks on a **revenue figure**, it should show the **detailed list of transactions** that make up that amount.

### 7. Detailed Reviews / Ratings
- Implement a **review system** similar to Shopee's, where users can rate specific aspects of the service.
- Example categories for a pet care platform: **Quality of Care**, **Grooming**, **Fragrance**, **Staff Behavior**, etc.

### 8. Two-Step Verification (2FA)
- If feasible, enable **two-step verification for every account**.

---

## Documentation

### 9. User Manual
- Create a **user manual** for the admin panel covering all major features and workflows.
